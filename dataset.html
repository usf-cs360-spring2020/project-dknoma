Dataset: Steam Reviews Dataset

Collection of Steam's best selling games reviews

This dataset isn't a comprehensive list of every single review of every single game on Steam as that would be gigabytes or terabytes of data. This dataset includes data only of the best selling games on Steam.

Info

This dataset contains 8 columns and 434,891 rows of data.

Columns

  1. Date Posted
    • Posting date of the review.
  2. Funny
    • Count of how many users found this review funny.
  3. Helpful
    • Count of how many users found this review helpful.
  4. Hours Played
    • How many hours the reviewer has played the game for.
  5. Is Early Access
    • If the game was in early access at the time of the review.
  6. Recommendation
    • If the reviewer recommends the game or not.
  7. Review
    • The user's review of the game.
  8. Title
    • The tile of the game.
