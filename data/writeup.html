Info and Conclusions

Data

This data set contains data of all SF311 cases created since 7/1/2008 with location information.

"This data is made available under the Public Domain Dedication and License v1.0 whose full text can be found at: www.opendatacommons.org/licenses/pddl/1.0/"

Resource retrieved from https://data.sfgov.org/City-Infrastructure/311-Cases/vw6y-z8j6

Columns

  1. requested_datetime
  2. service_name
  3. lat
  4. long
  5. point
  6. status_description

Wrangling

This data was imported and wrangled using the Socrata Open Data API (SODA). The data was limited to those that satisfied the following conditions:

  • Actively Open
  • Category: Parking Enforcement
  • Opened only within the month of February 2020
  • Point is not null

Acknowledgements

Additional resources used: https://data.sfgov.org/dataset/sfpd_districts/q52f-skbd
